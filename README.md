# Wallstreet

Choose your stocks -> Track your progress -> Learn and have fun

Real stocks, real data, play money!

This application is currently in active development. To try out the current version (pre-release) vist [Heroku](https://wallstreet-game.herokuapp.com/).

To get started in development:

  * Clone this repo
  * Install dependencies with `mix deps.get`
  * Create and migrate your database with `mix ecto.setup`
  * Install Node.js dependencies with `cd assets && npm install`
  * Start Phoenix endpoint with `mix phx.server`

Now you can visit [`localhost:4500`](http://localhost:4500) from your browser.
